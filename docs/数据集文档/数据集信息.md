## 数据集统计

### 语料库
- **文档总数**: 991 个
- **格式**: 每个文档包含 `title` 和 `text` 字段
- **示例**: 电影、人物、地点等维基百科风格文档

### QA数据
- **问题总数**: 100 个
- **格式**: 每个问题包含 `question`, `answer`, `relevant_docs`
- **相关文档**: 每个问题有多个相关文档标注（平均约1.5个）

## 数据格式示例

### 文档格式
```json
{
  "title": "Ed Wood (film)",
  "text": "Ed Wood is a 1994 American biographical period comedy-drama film..."

```

```json
{
  "question": "Were Scott Derrickson and Ed Wood of the same nationality?",
  "answer": "yes",
  "relevant_docs": [
    "Ed Wood (film)\nEd Wood is a 1994...",

```
